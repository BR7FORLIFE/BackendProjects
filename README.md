# BackendProjects

> Repositorio monorepo con proyectos backend enfocados en el ecosistema Spring Boot y NestJS.
> Cada carpeta es un proyecto independiente que explora distintos conceptos de arquitectura moderna.

---

## Proyectos

### 🏪 DistributorStore
Sistema de gestión de distribuidora. Proyecto enfocado en modelado de dominio y persistencia.

**Stack:** Spring Boot · PostgreSQL

---

### 🤖 IA-Websockets
Integración de IA con comunicación en tiempo real via WebSockets.

**Stack:** Spring Boot · WebFlux · WebSockets · Reactor Core

---

### 🗨️ mini-reddit
Mini plataforma tipo Reddit/Twitter. Arquitectura de microservicios con realtime, colas y notificaciones.

**Stack:** Spring Boot · NestJS · PostgreSQL · Redis · Kafka · RabbitMQ · Next.js

→ [Ver documentación completa](./mini-reddit/README.md)

---

### 🔐 nestjs-platform-auth
Plataforma de autenticación y registro de usuarios con manejo de excepciones.

**Stack:** NestJS · JWT · PostgreSQL

---

### ⚙️ started-with-microservices
Introducción práctica a microservicios con autenticación JWT y comunicación entre servicios.

**Stack:** Spring Boot · NestJS · Docker · JWT

## Conceptos practicados

- Arquitectura de microservicios
- Comunicación síncrona (HTTP) y asíncrona (Kafka, RabbitMQ)
- Autenticación con JWT y refresh tokens
- WebSockets y sistemas realtime
- Programación reactiva con WebFlux y Reactor
- Caché distribuida con Redis
- Colas y procesamiento background con BullMQ
- Containerización con Docker y Docker Compose
